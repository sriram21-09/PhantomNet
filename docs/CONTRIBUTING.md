# Contributing to PhantomNet

## Getting Started
1. Fork the repository
2. Create a feature branch: `git checkout -b [role]/[feature-name]`
3. Make your changes
4. Commit with descriptive message: `[Role] type: description`
5. Push and create a Pull Request

## Code Style
- Python: PEP 8 with Black formatting
- JavaScript: Airbnb style guide

## Testing
All PRs must include tests and maintain 80%+ coverage

## Pull Request Process
1. Update README if needed
2. Update documentation
3. Add tests for new features
4. Ensure all tests pass
5. Get approval from team lead
