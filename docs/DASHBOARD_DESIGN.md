# PhantomNet Dashboard Design

## Overview
The dashboard provides a high-level view of honeypot activity
using summary metrics, recent events, and attack distribution.

## Layout
- Metric cards at the top
- Events table on the left
- Attack chart on the right

## Components
- MetricCard: Displays key metrics
- EventsTable: Lists recent events
- AttackChart: Visualizes attack data
- Sidebar: Navigation menu

## Pages
- Dashboard
- Events
- Honeypots
